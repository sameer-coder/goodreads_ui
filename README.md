<h2 align="center">Install</h2>

Install with npm:

```bash
npm install
```

Install with yarn:

```bash
yarn install
```

<h2 align="center">Running the application</h2>
Run with the following command
```bash
npm start
```
React app server listens on port 3000

<h2 align="center">Technologies Used</h2>
- React
- Semantic UI (Styling)
- React router
- Redux
- Axios (Http client)
- ESLint (Linting code)
